Steps to update to the latest drop
1. Install perl if you haven't: http://strawberryperl.com/
2. Create a folder called LatestDrop in this directory
3. Unzip the latest loc drop zip to the LatestDrop directory
4. open all loc.xml files in this directory for edit
5. run copyLatestDrop.cmd from this directory
6. test, codereview, check in
